# Achaea-Wiki-DarkMode
A style to turn the Achaea Wiki to Dark Mode

# How to use
1. You need the "Stylus" extension. You can get it for [Chrome, Edge](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne), [Firefox](https://addons.mozilla.org/en-US/firefox/addon/styl-us/), and [Opera](https://www.bing.com/search?q=stylus%20extension%20opera&qs=n&form=QBRE&sp=-1&ghc=1&lq=0&pq=stylus%20extension%20oper&sc=12-21&sk=&cvid=E8649D34AD4F43359893922F62D1B2D6), either from the previous links or their [official GitHub repo](https://github.com/openstyles/stylus/releases/tag/v2.3.17).
2. Open ["Style" above or click on this link](https://github.com/Memerlin/Achaea-Wiki-DarkMode/blob/main/style)
3. Copy raw file.
4. Open the [Achaea Wiki](https://wiki.achaea.com/Main_Page)
5. Write a new style for https://wiki.achaea.com (DELETE THE /Main_Page AT THE END OF THE LINK)
6. Paste.
7. Done!

From there, you can edit the colours however you want. If you'd like to contribute to this repo, please do!
